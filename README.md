# Exploring Keyword Co-occurrence Networks in Scientific Publications  
Onyinyechi Ivory Uche 

### Overview
keyword network analysis to discover patterns in academic research keywords. I used a dataset of keywords and built a weighted network to find which topics are the most central research themes and the strongest conceptual linkages between them.

### Repository 
* `data/keywords_data.csv`: Source dataset of articles and keyword columns.
* `Project 3.ipynb`: Python implementation using Pandas, NetworkX, and Matplotlib.
